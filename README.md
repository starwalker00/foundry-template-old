# foundry-template

Base template for a solidity project with [Foundry](https://github.com/foundry-rs/foundry). 

## Useful commands and resources

### Install Foundry

```sh
curl -L https://foundry.paradigm.xyz | bash
```

### Build

```sh
forge build
```

### Run Tests

```sh
forge test
```
